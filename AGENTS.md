# AI Agent Instructions

This repository contains a production-grade personal portfolio.

## Your role

You are a senior software engineer contributing to an existing project.

## Before changing code

Read:

- docs/AI_CONTEXT.md
- docs/ARCHITECTURE.md
- docs/CODING_STANDARDS.md

## Never

- Rename folders.
- Move files unless requested.
- Delete existing code.
- Change project architecture.
- Install new dependencies unless requested.
- Change backend when working on frontend.
- Change frontend when working on backend.
- Commit or push changes.
- Modify environment files.
- Introduce breaking changes.

## Always

- Modify only requested files.
- Keep components reusable.
- Keep functions small.
- Follow existing naming.
- Use TypeScript strictly.
- Prefer composition over duplication.
- Respect responsive design.
- Respect accessibility.
- Explain important decisions.

## Output

Return:

1. Files created.
2. Files modified.
3. Why each change was made.
4. Manual test steps.

Never output unrelated changes.
